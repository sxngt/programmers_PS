갑자기 뭐 명함 하나를 회전해서 더 작아질수도 있다 이런 잡소리를 한줄 넣어놓자마자 문제가 갑자기 어떻게 구현해야할지 머리가 아파지기 시작하는데...

아무 의미가 없다는 뜻이 되는 것이지요.

명함은 일정한 가로와 세로의 길이를 가집니다. 그 말은 즉슨 정사각형이거나, 최소한 둘 중 긴 길이가 있다는 것이고, 우리는 긴 길이를 기준으로
일정하게 겹쳐놓아야지 최적으로 모든 명함을 담을 수 있는 최소길이를 찾을 수 있을것입니다.
